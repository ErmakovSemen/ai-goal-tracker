Пункт 6 — Соответствие тренера и цели/пользователя (итог)
Контекст:
Нужно определить, где хранится выбор тренера и пола, и как работает дефолт для новых целей.
Принятые решения
1) Тренер хранится в Goal
Каждый Goal имеет своего тренера
Тренер цели не зависит от профиля пользователя
2) Если тренер не выбран при создании цели — берём дефолт пользователя
Значения берутся из User (default_trainer_id, default_trainer_gender)
Это удобно и не требует ручного выбора каждый раз
3) Изменения всегда локальные
Смена тренера в Goal влияет только на эту цель
Смена дефолта в User влияет только на будущие цели
4) Пол тренера — отдельное поле
Хранится как trainer_gender
Используется только для формы речи и визуала
Не влияет на поведение тренера
Итоговая логика наследования
Goal.trainer_id задан → используем его
иначе → используем User.default_trainer_id
пол работает по той же логике
