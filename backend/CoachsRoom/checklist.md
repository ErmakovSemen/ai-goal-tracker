# Trainer Implementation Checklist

- [ ] Концепция тренера (поведение vs план)
- [ ] Перечень ключевых фич тренеров
- [ ] Формат данных тренера (JSON‑контракт)
- [ ] Структура БД под тренеров
- [ ] Стратегия ассетов и future‑rig
- [ ] Соответствие тренера и цели/пользователя
- [ ] Логика общения тренера в чате
- [ ] Логика напоминаний от тренера
- [ ] Подготовка JSON‑примеров (starter set)
- [ ] Наполнение БД начальными записями
- [ ] Заглушки ассетов (2 пола)
- [ ] UI: выбор тренера и пола
- [ ] Интеграция выбора в создание цели
- [ ] Связь тренера и задач/целей
- [ ] Интеграция в промпт генерации плана
- [ ] Интеграция в проактивные уведомления
- [ ] Валидация стиля (защита от “ломки характера”)
- [ ] Документация: как добавлять новых тренеров
- [ ] Дополнительные фичи (идеи и бэклог)
