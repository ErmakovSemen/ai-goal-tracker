# Рекомендуемые модели Ollama (по размеру)

## Легкие модели (для быстрого старта)

### 1. phi3 (~2.2 GB) ⭐ Рекомендуется
```bash
ollama pull phi3
```
**Плюсы:** Быстрая, хорошее качество для MVP

### 2. qwen2.5:1.5b (~1 GB) - Самая легкая
```bash
ollama pull qwen2.5:1.5b
```
**Плюсы:** Очень маленькая, быстрая

### 3. gemma2:2b (~1.4 GB)
```bash
ollama pull gemma2:2b
```
**Плюсы:** От Google, хорошее качество

## Средние модели

### 4. llama3.2 (~2 GB) - Текущая
```bash
ollama pull llama3.2
```
**Плюсы:** Хороший баланс качества и размера

### 5. mistral (~4.1 GB)
```bash
ollama pull mistral
```
**Плюсы:** Лучшее качество, но больше размер

## Как поменять модель

Просто измените в `backend/.env`:

```env
LLM_MODEL=phi3  # или qwen2.5:1.5b, gemma2:2b, и т.д.
```

И перезапустите бэкенд.

## Удаление ненужных моделей

```bash
# Посмотреть все модели
ollama list

# Удалить модель
ollama rm llama3.2
```

## Рекомендация для MVP

Используйте **phi3** - хороший баланс между размером и качеством.
