# AI Goal Tracker

AI-powered goal tracking application with chat-based goal formalization and progress monitoring.

## Features

- Chat-based goal formalization with AI
- Progress tracking with milestones
- Daily progress reporting
- Statistics and analytics
- Cute mascot for motivation
- Mobile application (APK available)

## Tech Stack

- Backend: Python/FastAPI
- Frontend: React/TypeScript
- Database: PostgreSQL

## Project Structure

- `/backend` - Python/FastAPI backend application
- `/frontend` - React/TypeScript frontend application

## Mobile Application

This project includes automatic APK generation using Capacitor and SourceCraft CI/CD.

### Building APK

The APK is automatically built on every push to the main branch. You can find the build instructions in [APK_BUILD_INSTRUCTIONS.md](APK_BUILD_INSTRUCTIONS.md).

### Testing

The latest APK can be downloaded from the CI/CD artifacts section after the build completes.
