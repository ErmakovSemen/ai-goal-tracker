# Настройка Together AI

## Быстрый старт

### 1. Регистрация и получение API ключа

1. Перейдите на https://together.ai
2. Зарегистрируйтесь (бесплатно)
3. Получите **$25 бесплатных кредитов** при регистрации
4. Перейдите в раздел **API Keys** в настройках
5. Создайте новый API ключ
6. Скопируйте ключ (начинается с букв/цифр)

### 2. Настройка в проекте

Откройте файл `backend/.env` и добавьте ваш API ключ:

```env
LLM_PROVIDER=together
LLM_API_KEY=your_together_api_key_here
LLM_MODEL=meta-llama/Llama-3-8b-chat-hf
```

### 3. Перезапустите бэкенд

После добавления ключа перезапустите бэкенд:

```bash
cd backend
# Остановите текущий процесс (Ctrl+C)
source venv/bin/activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

## Доступные модели

Together AI поддерживает множество моделей:

- `meta-llama/Llama-3-8b-chat-hf` (рекомендуется, по умолчанию)
- `meta-llama/Llama-3-70b-chat-hf` (более мощная)
- `mistralai/Mixtral-8x7B-Instruct-v0.1`
- `Qwen/Qwen2.5-7B-Instruct`
- `deepseek-ai/deepseek-coder-6.7b-instruct`

Чтобы использовать другую модель, просто измените `LLM_MODEL` в `.env` файле.

## Тестирование

После настройки протестируйте:

```bash
cd backend
./test_api.sh chat
```

Бот должен отвечать через Together AI вместо заглушки.
